- L1 全量分组注入（80轮，按日期标记）

## 配置调整建议

| 场景 | l1_inject_rounds | l1_save_rounds | 说明 |
|------|-----------------|----------------|------|
| 默认 | 80 | 200 | 平衡 token 和记忆完整性 |
| 轻量 | 20 | 100 | 节省 token，短对话场景 |
| 全量 | 200 | 500 | 深度任务，不差 token |
| 纯 L2+L3 | 0 | 200 | 不注入 L1，只用概括记忆 |
